Predicting the customer churn in the telecommunication industry that uses random forest algorithm. Random forest is popular for regression and classification tasks.

The data set that is used for the project is "Predicting Customer churn" data set that is obtained from IBM Watson Studio. The data set consists of 7043 observations with 21 variables that include customer demographic information, customer service information and customer account information. 

The variables that are included in this data set are account information, usage patterns, customer demographics and the customer churn status.
